BOYAS

Variedad de contenedores de pastillas para todo tipo de piletas.

  • BOYA CISNE

    Boya decorativa para pastillas de 200gr, con un gran espesor de plastico.

  • BOYA DOSIFICADORA BI COLOR

    Boya para pastillas de 200gr, con regulador para determinar el flujo del cloro.

  • BOYA DOSIFICADORA BI COLOR CON TERMOMETRO

    Boya para pastillas de 200gr, con regulador para determinar el flujo del cloro y con un termometro para medir la temperatura del agua.

  • BOYA HONGO MINI

    Boya para pastillas de 50gr, economica e ideal para piletas de lona.

  • BOYA HONGO

    Es una boya para pastillas de 200gr, economica y durable.

  • BOYA NINGBO

    Es una boya para pastillas de 50gr, de gran durabilidad por su espesor de plastico y con regulador del flujo de cloro, ideal para piletas de lona.

  • BOYA ROSETA

    Boya para pastillas de 200gr, con forma decorativa de rosa.

  • BOYA SATELITE CHICA

    Es una boya para pastillas de 50gr, con un aro que le permite tener una flotabilidad muy estable, ideal para piletas de lona.

  • BOYA SATELITE GRANDE

    Es una boya para pastillas de 200gr, con un aro que le permite tener una flotabilidad muy estable.
